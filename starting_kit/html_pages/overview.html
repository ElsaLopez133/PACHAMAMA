Living species classification problem

This is a flower classification problem. The data set contains 102 classes between 40 and 250 instances each, where each class refers to a type of flower.

References and credits:
Automated flower classificationover a large number of classes (https://www.robots.ox.ac.uk/~vgg/publications/2008/Nilsback08/nilsback08.pdf)

The starting kit was adapted from an Jupyper notebook designed by Balazs Kegl.
